## Methodology

**Data Sources:**
- CNCF Landscape: [@cncf/landscape](https://github.com/cncf/landscape)
- Homebrew Formula Search: `brew search` command
- Project categorization: Official CNCF maturity levels

**Analysis Approach:**
1. Extracted all graduated and incubating projects from CNCF landscape
2. Searched Homebrew formulae for each project by name
3. Verified package availability and captured formula URLs
4. Generated comprehensive tables with links to packages

**Limitations:**
- Some projects may have Homebrew packages under alternative names
- Homebrew cask packages are not included in this analysis
- Third-party taps are not covered
